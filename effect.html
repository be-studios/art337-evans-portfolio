Triangulate Effect

This is a generative and interactive video effect coded in resolume wire for use in avenue. The triangulate node is modified with 2 random options for scale. One of the options is completely random. Users can control the speed of the other option. Additionally, the levels inlet runs through the values 2, 4, 6, 8, and 10 before repeating. Users have the option to flip the effect. Finally, the output from the triangulate node goes through the suckr node where users can control the suck level with a float-in.
